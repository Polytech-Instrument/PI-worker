# Политех Верстальщик Web

Простая JS-версия генератора.

## Запуск

```powershell
cd C:\Users\Anton\Documents\Figma\web-generator
npm.cmd install
npm.cmd run dev
```

Открыть:

```text
http://localhost:3080
```

## Что Сейчас Есть

- `src/index.js` - один простой сервер на JS.
- `src/allProducts.json` - локальная база товаров, скопирована из Figma-плагина.
- `src/templates/index.html` - web-страница для просмотра данных.
- `src/templates/styles.css` - стили страницы.
- `/api/sheets` - список листов Google.
- `/api/data?gid=...&title=...` - данные выбранного листа:
  - сколько строк в CSV;
  - сколько строк с SKU;
  - сколько товаров найдено в JSON;
  - какие SKU не найдены;
  - разбивка по листовкам;
  - первые 20 карточек для просмотра.

## Telegram

Бот запускается из того же `src/index.js`.

1. Запускаешь сервер.
2. Пишешь боту `/start`.
3. Бот показывает листы Google таблицы.
4. Жмешь лист.
5. Бот присылает сводку по данным.
